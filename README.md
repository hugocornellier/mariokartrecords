# Chat App - TypeScript + React + Socket.io

## Dev:
- ```npm i && npm i --prefix ./frontend```
- ```npm start```
- In separate terminal:
- ```npm start --prefix ./frontend```

## Production:
- At root: ```npm run build```
- ```serve -s -n build```
- ```forever stopall; forever start ./backend/server.js```
